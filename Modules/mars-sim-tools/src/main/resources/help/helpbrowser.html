Help Browser

The Help Browser provides an inline hyperlink documentation about mars-sim. It doubles as a functional browser for viewing online resources.

(A). Shortcut Buttons

These buttons near the top of the Help Browser can take you quickly to Tutorial, User Guide, About page, Keyboard Shortcuts, as well as the online resources such as Project Website and Discussion Forum.


(B). Address Bar

The address bar displays the local path of the html page you are reading or the hyperlink of the website you're visiting.


(C). Navigation Buttons

The '<' button (or backspace) returns you to the page you previously visited.

The 'Go' button reloads the link on the address bar.

The '>' button sends you back to the next page that you have visited before you hit the '<'.

Note: normally, the page you are currently on is saved as the 'next' page for you to returns to whenever you hit '<'.


(D). Status Bar

If your mouse hovers over a hyperlink, its local path or url link will be printed on the status bar.
