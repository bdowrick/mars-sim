Welcome to the tutorial of the Mars Simulation Project (mars-sim) !

This tutorial comprises 12 steps, mostly focused on the usage of various Tool. Through these steps you will become familiar with the basics of viewing and interacting with the simulation via those tools.

Expect it to take about 10-15 minutes to read through this tutorial and recognizing the tools.

The tutorial is supplemented with a more in-depth User Guide, which explains various design concepts of mars-sim.

The User Guide is also accessible from the Help Browser by clicking its button.

You may switch back to the Help Tab at any time to resume the tutorial in the Help Browser.
