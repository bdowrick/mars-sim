Tutorial - Step 2 : The Main Simulation Window

The Main Simulation Window is the overall window area upon which the player may query how the simulation is doing.

Besides the main window, you should see a terminal or a console window, which houses the MarsLink A.I. that provides text interactions with players.

In window mode, you will see a row of icons right below title bar.

The desktop area hosts the content of any tools you may open.

At the start of a new simulation, it defaults to opening the Help Browser.

If the Sandbox Mode is chosen, no other tools will be popped open. In this case, the Monitor Tool will the status of each settlement in the simulation.
